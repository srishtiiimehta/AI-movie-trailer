## 📌 About the Project

**AI Movie Trailer Generator** is a Python-based project that uses Artificial Intelligence to automatically create movie trailers. Given a movie concept or description, the system handles everything — writing the script, generating visuals, editing the video, and producing voiceover audio — end to end.

---

## ✨ Features

- 🖊️ **AI Script Writing** — Automatically generates a compelling movie trailer script
- 🎥 **Text to Video Generation** — Converts script/prompts into video scenes
- ✂️ **Auto Video Editing** — Sequences and edits clips into a trailer format
- 🔊 **Voice/Audio Generation** — Generates voiceover and background audio
- 🚀 **End-to-End Pipeline** — Just input a concept and get a full trailer!

## ⚙️ How to Run

### 1. Clone the repository
```bash
git clone https://github.com/srishtiiimehta/ai-movie-trailer.git
cd ai-movie-trailer
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the project
```bash
python main.py
```
